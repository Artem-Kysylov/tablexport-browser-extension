🚀 TableXport: Новые кнопки экспорта

Встроенные прямоугольные кнопки для надёжного экспорта таблиц из AI чатов

❌ Старый подход Проблемы

  • Круглые кнопки вне таблицы
  • Абсолютное позиционирование
  • Обрезание в контейнерах
  • Только иконка + tooltip

✅ Новый подход Решение

  • Прямоугольные кнопки перед таблицей
  • Не ломает структуру данных
  • Никогда не обрезается
  • Иконка + текст

🎯 Ключевые улучшения (v2)

  • 🔧 Исправленная иконка: Batch кнопка использует ту же иконку что и одиночная
  • 📊 Сохранение структуры: Кнопка рендерится ПЕРЕД таблицей, не ломает данные
  • 🛡️ Надёжность: Кнопка всегда видна, никогда не обрезается
  • 🎨 Консистентность: Дизайн соответствует popup расширения (border-radius: 8px)
  • 💡 UX: Понятный текст вместо только иконки
  • ✨ Hover эффект: Зелёная тень при наведении
  • 🌐 Кроссплатформенность: Работает одинаково на всех AI чатах
